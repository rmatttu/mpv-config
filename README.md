# mpv-config

mpv config files

## Install

* Download mpv from [mpv.io | Installation](https://mpv.io/installation/)

### Windows

```bat
cd %AppData%
git clone https://github.com/rmatttu/mpv-config.git mpv --recursive
```

### scoop

```bash
scoop bucket add extras
scoop install mpv
./copy-settings.sh ~/scoop/persist/mpv/portable_config
```
